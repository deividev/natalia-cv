# 🎉 BIENVENIDO A TU LANDING PAGE DE SEGUROS

Esta es una template profesional, moderna y **100% personalizable** para asesores de seguros de salud.

---

## ⚡ INICIO RÁPIDO (5 MINUTOS)

### 1️⃣ Abre el archivo: **`config.js`**

### 2️⃣ Cambia estos valores:

```javascript
personal: {
    nombre: "TU NOMBRE AQUÍ",
    telefono: "34XXXXXXXXX"  // Tu WhatsApp
}
```

### 3️⃣ Guarda y abre **`index.html`** en tu navegador

**¡Ya tienes tu web personalizada!** 🎊
