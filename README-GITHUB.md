# 🏥 Landing Page Template - Seguros de Salud

Template profesional, moderna y totalmente personalizable para asesores de seguros de salud. Solo editas un archivo de configuración y tienes tu web lista.

## ✨ Características

- 🎨 **Diseño moderno y profesional**
- 📱 **100% responsive** (móviles, tablets, desktop)
- ⚡ **Fácil personalización** - Solo un archivo de configuración
- 🚀 **Sin instalación** - Abre y funciona
- 💬 **Integración con WhatsApp** - Botones estratégicos
- 🌐 **Lista para publicar** - Compatible con GitHub Pages, Netlify, Vercel
- 🎯 **Optimizada para conversión** - Diseño enfocado en captar clientes

---

## 🚀 Inicio Rápido

### 1. Descarga o clona este repositorio

### 2. Abre `config.js` y edita tu información:

```javascript
const CONFIG = {
  personal: {
    nombre: "Tu Nombre",
    telefono: "34XXXXXXXXX", // WhatsApp
    // ...
  },
};
```

### 3. Abre `index.html` en tu navegador

**¡Listo!** Tu web está personalizada 🎉

## 🌐 Publicar tu Web

### Opciones Gratuitas:

#### GitHub Pages

```bash
1. Crea un repositorio en GitHub
2. Sube tus archivos
3. Ve a Settings → Pages
4. Selecciona la rama main
5. ¡Listo! Tendrás una URL: username.github.io/repo
```

#### Netlify

```bash
1. Ve a netlify.com
2. Arrastra tu carpeta
3. ¡Publicado! URL automática
```

#### Vercel

```bash
1. Ve a vercel.com
2. Importa tu proyecto
3. Deploy automático
```
